# [인력양성] 건강 리포트 📈

건강 리포트는 대조군, 중재군A, 중재군B 전체 군 공통 제공용이며, 초기 건강 설문 응답결과를 바탕으로 결정되어 12주 실증 기간동안 고정됩니다.

**데이터:**
- 건강 검진: 강남검진센터 연동 (허리둘레, 혈압, 공복혈당, 중성지방, HDL-C)
- 건강 문진 (식습관 설문 19문항 + 생활습관 설문): 현재 텍스트 기반 → 추후 이미지 기반 입력으로 전환 예정 (서봉원 교수님 연구실 작업 필요)


**알고리즘 파이프라인:**
```
1. 초기 건강 설문 응답 수집
2. 건강 설문 응답 분석
   2.1. Auto-encoder를 통한 잠재 벡터 추출
   2.2. K-Means 클러스터링 → 식습관 유형 배정
   2.3. SHAP 분석 → 개인별 대사증후군 기여 위험요인 도출
   2.4. 위험요인 순위화 (4개 요인: 식습관 클러스터, 음주, 흡연, 신체활동)
3. 네트워크 분석 결과를 이용한 위험(허브식품) 식습관 도출
   - 성별 × 연령대 × 대사증후군 유무별 층화 그룹의 허브식품 매핑
4. 시각화를 위한 결과 반환
   4.1. 19개 식습관 종합 평가 (레이더 차트)
        - 식습관 응답 (나의 식습관): 19개 문항
        - 위험(허브식품) 식습관: 상위 5개 (top-5)
   4.2. 대사증후군 구성요소별 위험도 표시
        - 5개 지표 각각의 현재 수치 및 정상/위험 판정
```

**참고:** 건강 리포트 UI/UX

`건강 리포트.html`을 함께 보내드립니다. `run_report.py`에서 쓰이는 변수들이 어떻게 시각화되면 좋을지 **레이아웃 참고용**으로 봐주시면 됩니다.
